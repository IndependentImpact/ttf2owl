# Token Taxonomy Framework (TTF) repo notes

These notes summarize the public InterWork Alliance Token Taxonomy Framework repository
(https://github.com/InterWorkAlliance/TokenTaxonomyFramework) as reviewed for this task.

## Key repository sections (from README)

- **Taxonomy Overview** (`token-taxonomy.md`): overview and taxonomy syntax/definitions used to create artifacts.
- **Taxonomy Model** (`taxonomy-model.md`): technical overview of the model used to interact with the taxonomy.
- **Control Messages** (`token-control-messages.md`): message-based specifications used for certification.
- **Taxonomy Artifact Format** (`taxonomy-artifact-format.md`): instructions and format rules for taxonomy artifacts.
- **Artifacts** (`artifacts/`): sample/template artifacts for base token types, behaviors, and behavior groups.
